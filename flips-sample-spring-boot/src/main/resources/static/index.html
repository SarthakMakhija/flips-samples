Flips Sample (Spring Boot) Explore endpoints to understand the behavior of @Flip* annotations.
Loads All Articles
Endpoint /articles is annotated with -
  • @RequestMapping(value = "/articles", method = RequestMethod.GET)
  • @FlipOnProfiles(activeProfiles = "dev")
which will be enabled in dev profile.
Loads an article by Id
/articles/{id} Loads an article by Id.
Endpoint /articles/{id} is annotated with -
  • @RequestMapping(value = "/articles/{id}", method = RequestMethod.GET)
  • @FlipOnEnvironmentProperty(property = "feature.article.by.id", expectedValue = "Y")
which will be enabled if feature.article.by.id property is set with value Y.
Loads the article statistics
Endpoint /articles/statistics is annotated with -
  • @FlipBean(with = ArticleStatisticsController.class)
  • @FlipOnSpringExpression(expression = "@environment.getProperty('feature.movie.statistics') == 'Y'")
  • @RequestMapping(value = "/articles/statistics", method = RequestMethod.GET)
which will flip the implementation with ArticleStatisticsController if feature.movie.statistics property is set with value Y.
